noisee is the result of my thesis at <a href="https://unicornuniversity.net" target="_blank" rel="noopener noreferrer">Unicorn University</a>, which explores procerdurally generated gradient noise (such as <a href="https://en.wikipedia.org/wiki/Perlin_noise" target="_blank" rel="noopener noreferrer">Perlin Noise</a>). It is an interactive visual interface which allows users to experience different uses of gradient noise and experiment with different settings of the noise generator.
